# Transfer Track Agent Guidance

This repository is a hackathon MVP. Future agents should follow these rules:

- Prioritize the demo flow over feature breadth.
- Do not add authentication, Supabase, or a real backend.
- Do not build chat, comments, likes, notifications, or a social feed.
- Use mock data and localStorage for student profiles and interest requests.
- Keep UI and code simple, readable, and focused on onboarding, matching, and professor discovery.
- Use React Router for pages and Tailwind CSS for styling.
